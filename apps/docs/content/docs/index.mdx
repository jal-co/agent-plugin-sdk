---
title: Introduction
description: Write an agent plugin once, ship it to every harness.
---

`agent-plugin-sdk` lets you define a plugin **once** in TypeScript and compiles
it to the **native installable artifacts** each harness expects — no runtime, no
wrapper. The output is exactly the files those harnesses load on their own.
